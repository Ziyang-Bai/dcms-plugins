Модуль автоматической покупки рекламы через ВебМоней Мерчант.
Скрипт написан на pdo.
Автор Sakamsky99.
Права на продажу принадлежат Лиле(красота) сайт Соблазняшки.Рф
Модуль Запрещено перепродавать, дарить друзьям, кидать в пабл и тому подобное.

Установка:
1) Распаковать в корень
2) Перейти в сервис  и на https://merchant.webmoney.ruстроить свой кошелек, 
   как показано на скрине skrin.png (расположен в папке WebMoney, то есть в папке скрипта)
3) Перейти в файл /WebMoney/core.php и прописать данные от базы данных. 
   Настроить там же сам скрипт, то есть вписать свой wmr кошелек, secret key, 
   настроить цену. в файле rek.php прописать Ваш кошелек.(все действия в файлах закоментированы)
4) Выполнить запрос в базу из файла dump.sql
5) Проинклудить файл /WebMoney/reklama.php в удобное для вас место например в foot вашей темы

echo '<img src="/style/icons/rek.png">' ;
echo "<a href='/WebMoney/'><span style='color:blue'>[+] Купить Рекламу</span></a></br>" ;
require_once ('WebMoney/reklama.php');

6) В файле adm_panel/index.php прописать ссылку на админку управления рекламой

if ($user['id']==1)echo "<div class='main'><img src='/style/icons/str.gif' alt=''/><a href='/WebMoney/admin.php'>Реклама Админка</a></div>\n";
   вывод сделан на id 1 вы можете прописать как хотите,доступ в админку по паролю который вы указали
   в файле core.php
