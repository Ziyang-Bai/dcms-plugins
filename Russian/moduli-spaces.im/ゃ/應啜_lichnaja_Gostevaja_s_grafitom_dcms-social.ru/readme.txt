Название: Личная гостевая пользователей
Автор: Killer (http://about.lovkins.ru)
Версия: 1.1
Требования: DCMS 6.6.4 (Fiera, Social), PHP 4.2

### Описание ###
Этот скрипт позволит иметь пользователям вашего сайта гостевую книгу с большим функционалом.
Ну, как вы уже догадались, основной функцией этого скрипта есть отправка сообщений в гостевую другим пользователям.
При отправке сообщения к нему можно прикрепить файлы, а для тех кто с ПК еще и граффити.
Если файл является изображением или видео-файлом, то к нему автоматически генерируется скриншот. Если человек с телефона, то генерируется маленькое превью (80х80), если с ПК - большое (200х200).
Для звуковых файлов выводится плеер, с помощью которого можно прослушать полученную песню в онлайне!
Если пользователь закончил рисовать граффити и нажал копку "Отправить", то оно автоматически попадает в список прикрепляемых фалов к новому сообщению в виде файла изображения.
Максимальное к-тво прикрепляемых файлов - 5 (включаю граффити; можно настроить отредактировав только один файл - configs.php!).
Пользователь может настроить доступность своей гостевой книги и кто может оставлять в ней комментарии. Смена доступности гостевой стоит 50 баллов (тоже можно настроить).
Есть удаление сообщений, их редактирование, ответ на них, скрытие/показ. Так же на сообщения можно подавать жалобы, которые отправляются в почту Администрации ресурса.
Есть уведомления о новых сообщениях в гостевой, которые приходят ее владельцу и всем, кто оставил в ней сообщения. При этом, если такое уведомление уже есть, то не отправляется еще одно, а редактируется старое - изменяется время отправки и, если прочитано, становится не прочитанным!
Это все. Будут обновления!

### Установка ###
1) Папку us_guest закинуть в папку user.
2) Выполните запросы из файла us_guest.sql.
3) На папки user/us_guest/files и us_guest/screens поставить chmod 777.
4) В файле info.php прописать код:
echo "<img src='/user/us_guest/images/guestbook.png' /> <a href='/user/us_guest/?user_id=$ank[id]'>Гостевая</a> (";
include('user/us_guest/count.php');
echo ")<br />\n";
5) На файл user/us_guest/inc/zip_images.zip поставить chmod 777.
6) Это все!
