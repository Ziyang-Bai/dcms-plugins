Установка мода Почта PRO 2.6.41) Залить в корень с заменой всё, что не в папке readme.2) Установить на папку mfiles права 777.3) Зайти в КАЖДУЮ тему, где вы хотите организовать работоспособность мода, и:      Файлы из темы (head.php и foot.php) заменить на аналогичные из папки readme мода.4) Открыть папку sys/inc/ и заменить файлы tfoot.php и thead.php на аналогичные из папки readme мода.5) Открыть папку sys/inc/ и заменить файл user.php на аналогичный из папки readme мода.      (ЕСЛИ У ВАС ФАЙЛ USER.PHP НЕ ОРИГИНАЛЬНЫЙ, ВСТАВЬТЕ В КОНЕЦ СВОЕГО ФАЙЛА ТО, ЧТО ОТМЕЧЕНО В МОЕМ)6) Открыть файл из мода readme/aut.php и почитать то, что выделелно комментариями.    Сделать то же самое в своем файле sys/fnc/aut.php7) Готово! Почта установлена и готова к работе.
